Selector :- Selecters are used to seletct the element to style

Types of Selector

  1. Id Selector
  2. Class Selector
  3. Element Selector
  4. universal Selector
  5. Group Selector
  6. Descendent Selector or
  7. Child Selector
  8. Adjacent Selector
  9. General Selector
  10. Attribute Selector

Use of ID Selector

ID Selector :- It have a unique identifier and it is a higher Specificity than other selectors. 
Syntax:- #id{
    property : value;

Universal Selector :- It is used to select all elements on the page. 
 Syntax:-  *{
    property : value;
    }

 Syntax:-  p, h1, h2{
    property : value;

Descendent Selector :- It is used to style both direct and indirect children of a specified element. 
 Syntax:-  div .p{
    property : value;

Child Selector :- It is used to style only direct children of a specified element. 
 Syntax:-  div > p{
    property : value;
    }
Adjacent Selector :- It is used to style an element that is immediately adjacent to another specified element. 
 Syntax:-  h1 + p{
    property : value;
    }
General Selector :- It is used to style all elements that are siblings of a specified element, regardless of their position. 
 Syntax:-  h1 ~ p{
    property : value;
    }
